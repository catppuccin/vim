## Usage

### Manually


1. Clone this repository
2. Create a folder inside your `.vim` named `colors`
3. Put the `catppuccin.vim` file inside `colors`
4. When Vim is running use `:colorscheme catppuccin` to apply the theme

### Vim-Plug
```
Plug 'catppuccin/vim'
```
